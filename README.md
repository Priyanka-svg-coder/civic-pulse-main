# Civic Pulse App 

## Overview

Civic Pulse is a frontend web application designed to provide users with information and updates related to civic activities and services.

## Features

* Interactive user interface
* Responsive design for mobile and desktop
* Dynamic content rendering
* Clean and user-friendly layout

## 🛠 Tech Stack

* React JS
* CSS
* Vite
